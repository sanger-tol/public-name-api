What are ToLIDs?

ToLIDs are unique IDs for individual specimen. They facilitate recognition and communication and support taxonomy context. For example, ilAlcRepa1 is "insect", "lepidoptera", "Alcis", "repandata", "first individual sampled". The ToLID API allows anyone to lookup ToLIDs and authorized users to create new ones.

How can I use them?

The best place to get started is the API documentation. To create ToLIDs, you'll need an API key from the ToLID team.

What if the species is missing in the ToLID database?

The ToLID database contains all British angiosperms (BotSocBritIsles), all valid British taxa with taxonomy (NHM) and all “species” with sequence in INSDC. If a species is not in the database, an error message will be returned by the API. Please contact us and we'll add it.
